About the Project

This project is an implementation of a wiki-lite page for imdb titles (shows and movies). Allowing users to easily browse data, and change the data if it is incorrect.

SQLite3 - IMDB DB

SQLite3 is a tool that can be used alongside Node.js to query a database with SQL. This allowed us to pull dynamic data from the IMDB's database of actors, shows, and movies.

Node.js - Express.js

Node.js is an open-source, cross-platform JavaScript run-time environment that executes JavaScript code server-side. This was used to execute commands with SQLite3 and serve files to the browser. Express.js is a web application framework for Node.js that allows for better management of requests made to the Node.js server.

Bootstrap4 - Angular.js - jQuery

This site was created using both Bootstrap 4 and AngularJS. Bootstrap was used to format the page correctly using their grid system to section the page and prvice some default CSS styling. Angular was used for sending requests to the Node.js server for dynamic to populate and edit the pages. We also used jQuery for manipulation of the DOM.
